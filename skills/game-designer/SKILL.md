# game-designer

## Purpose
Create universal assembly plans for interactive worlds and export them to supported runtimes.

## Workflow
1. Analyze the Universal Assembly Graph.
2. Validate nodes, connections and constraints.
3. Select the appropriate engine adapter.
4. Use available MCP tools for execution.
5. Verify exported digital twin results.

## qFoldIT Principle
One assembly graph, multiple runtimes.
