# qFoldIT Engine Integrations

qFoldIT uses an engine-neutral Digital Twin -> Scene Plan architecture.

## Supported targets

| Platform | Bridge | Status |
|---|---|---|
| NVIDIA Omniverse | NVIDIA-Omniverse/kit-usd-agents | Official NVIDIA USD/Kit agent workflow |
| Unreal Engine | EpicGames/unreal-engine-skills-for-claude-code-plugin | Official Epic Games Claude Code skills/plugin workflow |
| Unity | CoplayDev/unity-mcp | Community MIT MCP bridge, not affiliated with Unity Technologies |
| Unigine | unigine-engine/ai-docs | Documentation-driven AI workflow |

The qFoldIT core does not duplicate engine functionality. It provides
scientific molecular data and lets specialized engine bridges perform scene
creation.
